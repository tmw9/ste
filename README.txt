Title : Terminal Based Text Editor (ste - Small Text Editor)
MIS : 111603053

Description:
    This is a simple terimanl based text eidtor (ste).
    It has the following features:
        1. Navigating in files
        2. Can navigate to big files which dont fit in terminal.
        3. You can cut lines.
        4. You can paste lines.
        5. You can use erase characters using BACKSPACE
        6. Save files.
